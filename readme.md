# ?? Chef Bot - Мультиязычный кулинарный бот для Telegram

Умный бот-шеф, который помогает приготовить вкусные блюда из того, что есть под рукой. Поддерживает 6 языков, избранные рецепты, кэширование и аналитику.

## ?? Возможности

- **Мультиязычность**: Поддержка русского, английского, немецкого, французского, итальянского и испанского языков
- **Голосовой ввод**: Отправляйте продукты голосовым сообщением
- **Избранные рецепты**: Сохраняйте понравившиеся рецепты в базу данных
- **Умный анализ**: AI определяет, какие блюда можно приготовить из ваших продуктов
- **Кэширование**: Экономия запросов к Groq API
- **Аналитика**: Сбор метрик использования
- **Премиум функции**: Расширенные возможности для платных пользователей
